## requirements

- poetry (python package manager)
- misskey admin token

## install

1. `poetry install`

## usage

1. set environment variable (`MISSKEY_ADMIN_TOKEN`, `MISSKEY_HOST`)
2. execute fetcher.sh


e.g.

```shell-session
env MISSKEY_ADMIN_TOKEN=some_misskey_token MISSKEY_HOST=misskey.example.com ./fetcher.sh
```
